Setting Up Your Python Environment

Why Anaconda?

It includes Python, Jupyter, and key data science libraries (pandas, numpy, matplotlib) in one installation.

Installing Anaconda/Miniconda

Option 1: Anaconda (Recommended for beginners)

  1. Visit Anaconda Individual Edition
  2. Download the installer for your operating system
  3. Run the installer and follow the prompts
  4. Accept the default settings unless you have specific requirements

Option 2: Miniconda (Lighter installation)

  1. Visit Miniconda
  2. Download the installer for your operating system
  3. Run the installer and follow the prompts

Setting Up Jupyter Notebook Environment

Command Line Setup

  1. Open Terminal (Mac/Linux) or Anaconda Prompt (Windows)
  2. Create a new environment for the course: 
    conda create -n stat_computing python=3.9
  3. Activate the environment: 
    conda activate stat_computing
  4. Install required packages: 
    conda install jupyter pandas numpy matplotlib seaborn scikit-learn
  5. Launch Jupyter Notebook: 
    jupyter notebook

Verification

Test your installation by creating a new notebook and running:

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

print("Setup successful!")

Alternative: Google Colab

If you prefer not to install software locally, you can use Google Colab:

  1. Go to colab.research.google.com
  2. Sign in with your Google account
  3. Create a new notebook
  4. Most packages are pre-installed

Troubleshooting

  • Path issues: Make sure Anaconda is added to your PATH
  • Package conflicts: Create a fresh environment if you encounter issues
  • Jupyter not opening: Try python -m jupyter notebook
